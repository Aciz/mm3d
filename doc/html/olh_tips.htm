Quick Tips

This page describes some short tips that can help you use Misfit Model 3D more effectively.

General Modeling Tips

  • Use the properties panel
  • Background images
  • Snap to grid/vertex
  • Drag vertex on edge
  • Edge turn
  • Edge divide
  • Extrude
  • Holding shift tips
  • Normals face out
  • Snap together / weld / unweld

Viewports

  • Toggle ortho/persp viewport
  • Middle mouse pan
  • Scroll wheel zoom
  • Ctrl-Rotate with wheel and pan buttons
  • Ctrl-Rotate ortho
  • Viewport bookmarks
  • Frame selected/all

Texturing

  • Remember must have group for texture, all triangles share
  • Middle mouse pan
  • Scroll wheel zoom
  • Texture projections
  • Paint Texture
  • Reload Textures

Importing and Exporting

  • File | Export
  • Importing Animations
  • Merging
  • Quake MD2
  • Quake MD3

Animations

  • Use rotate instead of translate
  • Set keyframe in first and last frames
  • Use Set Rotation/Translation Keyframe for default starting point
  • Converting skeletal to frame

Alpha Blending

  • Disable back-facing polygons
  • Eliminate co-planar triangles (separate)

Advanced

  • Custom keyboard shortcuts
  • Boolean operations
