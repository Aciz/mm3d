Shininess Window

The Shininess Window is used to change light properties for a texture. It is made up of a slider and a text entry box for the shininess value. This window is accessible from the Materials Window.

The shininess property of a material ranges from 0 to 100. A higher shininess value means the material is more shiny (yeah, sounds obvious--but you never know). You can drag the slider left to decrease the shininess value, or drag it right to increase the shininess. You can also change the value by typing a new value into the text box next to the slider.

Press Ok to keep your changes or press Cancel to ignore any changes.
